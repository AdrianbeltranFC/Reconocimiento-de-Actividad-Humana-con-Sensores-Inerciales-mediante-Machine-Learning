# Reconocimiento-de-Actividad-Humana-con-Sensores-Inerciales-mediante-Machine-Learning
Repositorio que contiene el código para el Proyecto Integrativo II, enfocado en el procesamiento de datos de un sensor IMU (WT9011DCL) para la clasificación de tres actividades humanas: "Quieto", "Caminar" y "Correr".
